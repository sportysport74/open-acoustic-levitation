        │     - Emitter black (-) to MOSFET drain            │
        │     - Reversed = NO WORK                           │
        │                                                    │
        │ ⚠️  MOSFET ORIENTATION CRITICAL!                   │
        │     - Flat side facing you                         │
        │     - Left = Gate, Middle = Drain, Right = Source  │
        │     - Wrong = Magic smoke!                         │
        │                                                    │
        │ ⚠️  MUST HAVE PULL-DOWN RESISTORS!                 │
        │     - 10kΩ from each gate to ground                │
        │     - Missing = Floating gates = erratic behavior  │
        │                                                    │
        │ ⚠️  FUSE IS NOT OPTIONAL!                          │
        │     - Protects against shorts                      │
        │     - 2A in 12V power line                         │
        │     - Could save your house from fire!             │
